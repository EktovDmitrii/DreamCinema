# Movie-Time

 
## Описание: 
Все новинки и самые популярные фильмы у тебя в смартфоне. Movie-Time - это возможность каждый день получать самую свежую информацию из мира кино.


![READMEPHOTO](https://user-images.githubusercontent.com/117804166/207007019-77266a67-883d-494c-a21d-081e6febe229.jpg)


## Пользовательские возможности: 

<li> Следи за новинками каждый день </li>
<li> Сохраняй любимые фильмы в избранное </li>
<li> Уникальная галерея по жанрам </li>
<li> Детальная информация по каждому фильму, с возможностью просмотра трейлеров </li>
